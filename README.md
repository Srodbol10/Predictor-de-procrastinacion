# Predictor-de-procrastinacion

Aplicación web desarrollada con Streamlit que permite predecir el riesgo de procrastinación de un usuario a partir de sus hábitos diarios.

Este proyecto utiliza un modelo de Machine Learning (Regresión Logística) para analizar variables como:
 Horas de sueño
 Uso de redes sociales
 Nivel de energía
 Dificultad de la tarea
-----------------------
Con base en estos datos, el sistema:
Calcula la probabilidad de procrastinar
Muestra el nivel de riesgo (bajo, medio o alto)
Detecta factores clave
Genera recomendaciones personalizadas
Almacena los datos para mejorar el modelo
-----------------------
Tecnologías utilizadas:
Python
Streamlit
Pandas
Scikit-learn
Matplotlib
------------------------
Características principales:
Interfaz interactiva tipo app
Predicción en tiempo real
Sistema de recomendaciones
Visualización de datos
Almacenamiento de datos en CSV
Análisis opcional mediante gráficas
Aprendizaje del sistema
------------------------
La aplicación guarda los datos de cada usuario en un archivo .csv, permitiendo:
Construir una base de datos real
Mejorar el modelo con el tiempo
Analizar patrones de comportamiento
